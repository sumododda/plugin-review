## Checklist Items

### D9: Build, CI/CD & Release Integrity (weight: 7)

BLD-01 - CI workflow hardening:
- Read all files in {CLONE_DIR}/.github/workflows/ (if directory exists)
- Check if third-party actions are pinned to commit SHAs: rg -n "uses:.*@" {CLONE_DIR}/.github/workflows/
- Flag any uses: that reference a tag (v1, v2, latest) instead of a full SHA
- Check permissions: rg -n "permissions:" {CLONE_DIR}/.github/workflows/

BLD-02 - CI secrets safety:
- Search: rg -n "id-token.*write|oidc" {CLONE_DIR}/.github/workflows/
- Check for secrets exposure in CI logs

BLD-03 - Artifact provenance:
- Check for signing, attestation, SBOM in release workflow
- Search: rg -n "attest|provenance|cosign|sigstore|sbom" {CLONE_DIR}/.github/workflows/

BLD-04 - Reproducible builds:
- Check for build documentation, pinned toolchain versions

BLD-05 - Version pinning and updates:
- Check for CHANGELOG.md
- Check for version pinning documentation

### D10: Repository Health & Maintainer Trust (weight: 5)

REP-01 - Maintainer identity:
- Run: gh repo view {REPO_URL} --json owner,name,description,stargazerCount,forkCount,createdAt,updatedAt 2>/dev/null || echo "gh not available"
- Check if owner is a real person/org with public track record

REP-02 - Active maintenance:
- Run: git -C {CLONE_DIR} log --oneline --since="90 days ago" | wc -l
- Run: git -C {CLONE_DIR} log --oneline -1 --format="%ci" to get last commit date

REP-03 - Bus factor:
- Run: git -C {CLONE_DIR} shortlog -sn --no-merges | head -20
- Count unique contributors

REP-04 - Established history:
- Run: git -C {CLONE_DIR} log --oneline --reverse | head -1 (first commit date)
- Check repo age, commit count

REP-05 - AI-authored code:
- Run: git -C {CLONE_DIR} log --all --grep="Co-Authored-By\|co-authored-by\|Generated by\|AI generated\|Copilot\|Claude\|GPT\|Cursor" --oneline | head -10
- If AI co-authored code found, check for evidence of human review

REP-06 - Marketplace verification:
- Check for verified publisher badges (via gh if available)
- Run: git -C {CLONE_DIR} log --show-signature -3 2>/dev/null (signed commits?)

REP-07 - Branch protection:
- Run: gh api repos/{OWNER}/{REPO}/branches/main/protection --jq '.required_pull_request_reviews,.required_status_checks' 2>/dev/null || echo "Cannot check branch protection (gh auth or permissions)"

OPS-03 - Changelog:

OPS-04 - Incident response:

OPS-05 - Rollback/kill switch:

OPS-06 - CI security gates:
